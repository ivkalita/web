Наконец, функцию можно задать с помощью таблиц, т.е. для некоторых значений переменной x указать соответствующие значения переменной y. Данные таблиц могут быть получены как непосредственно из опыта, так и с помощью тех или иных математических расчетов. Примерами такого задания функций являются логарифмические таблицы и таблицы тригонометрических функций.
